# Github Dark Default CauseYNot
Modified version of the Github Dark Default theme that I use in VSCode/Cursor IDE.

### Installation:
- Place into `~/.vscode/extensions/github.github-vscode-theme-{version number}/themes/`
- Add into `github.github-vscode-theme-{version number}/package.json` like so:
```
      ...
      {
				"label": "GitHub Dark",
				"uiTheme": "vs-dark",
				"path": "./themes/dark.json"
			},
			{
				"label": "GitHub Dark Default CauseYNot",
				"uiTheme": "vs-dark",
				"path": "./themes/dark-default-causeynot.json"
			}
		]
...
```
